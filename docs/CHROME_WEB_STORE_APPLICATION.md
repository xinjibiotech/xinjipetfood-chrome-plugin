## Store Listing Copy

### Extension Name

Xinji Pet Food OEM Product Link

### Short Description

Xinji pet food OEM navigator for B2B buyers: search formulas, compare ingredients, export profiles and open official links.

### Detailed Description

Xinji Pet Food OEM Product Link is a lightweight Chrome extension for B2B pet industry buyers, distributors and sourcing teams who need faster access to verified Xinji product and cooperation resources.

The extension focuses on practical pet food OEM workflows:

- Search pet food products by formula, ingredient, category or packaging need.
- Open stable official Xinji website pages for pet food, OEM/ODM, pet supplements, animal health and pet export topics.
- Compare two product profiles by ingredients, category, packaging and official link.
- Save favorite product references for repeated buyer communication.
- Export product lists as CSV or JSON for internal sourcing records.
- Open an internal product detail fallback when a dedicated product page is not available.

This extension is built for B2B users who research pet food OEM, private label pet food, pet supplements and export-ready pet products. It connects product discovery with official Xinji Pet Food resources, including pet food categories, OEM/ODM cooperation, factory information, quality certification and contact pages.

## Privacy And Permission Answers

### Single Purpose

This extension helps B2B pet industry users search Xinji product references, compare product information, save favorites, export local product profiles and open verified official Xinji website pages.

### Permissions

Permission requested:

- `storage`

Justification:

The extension uses Chrome local storage only to save favorite products in the user's browser. It does not read browsing history, does not access page content and does not send saved data to a server.

### Host Permissions

None.

The extension opens official website links with normal browser navigation. It does not require content scripts or host access to `xinjipetfood.com`.

### Remote Code

No remote code.

All JavaScript, HTML, CSS and image assets are included in the extension package.

### Data Collection

Recommended dashboard selection:

- The extension does not collect user data.

Privacy statement:

The extension does not collect, transmit, sell or share personal data. Favorites are stored locally in the user's browser through Chrome local storage. CSV and JSON files are generated locally from product information displayed in the extension.

## Review Risk Notes

Higher approval probability:

- Official Xinji account submits the extension.
- Listing states a clear single purpose.
- Permissions stay minimal.
- Privacy policy matches actual behavior.
- Screenshots show real user-facing functionality.

Main rejection risks:

- Submitting from a non-official account while using Xinji name and logo.
- Describing the extension primarily as an SEO/backlink tool.
- Claiming product data that is not visible or approved on the official website.
- Missing privacy policy URL.
- Promotional images with too much text or misleading claims.
